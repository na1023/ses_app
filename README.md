# SES業務管理アプリケーション

Streamlit + pandas + CSV によるローカル完結型の SES 業務管理ツールです。

## ディレクトリ構成

```
ses_app/
├── app.py                    # エントリーポイント（ダッシュボード）
├── requirements.txt          # 依存ライブラリ
├── .streamlit/
│   └── config.toml           # テーマ設定
├── pages/
│   ├── 1_案件管理.py          # 案件の登録・編集・削除
│   ├── 2_面談_ToDo管理.py    # 面談・ToDoの管理
│   ├── 3_日報管理.py          # 日報の入力・閲覧
│   ├── 4_レポート.py          # 週報・月報・グラフ
│   └── 5_設定_データ管理.py   # インポート・エクスポート・バックアップ
├── utils/
│   ├── __init__.py
│   ├── data_manager.py       # CSV 読み書き・バックアップ・初期化
│   └── styles.py             # 共通 CSS・バッジ関数
├── data/                     # CSV データ保存先（自動生成）
│   ├── projects.csv
│   ├── interviews.csv
│   ├── todos.csv
│   └── daily_reports.csv
└── backup/                   # 自動バックアップ先（自動生成）
    └── YYYYMMDD_HHMMSS/
```

## セットアップ・起動方法

```bash
# 1. 依存ライブラリをインストール
pip install -r requirements.txt

# 2. アプリを起動
streamlit run app.py
```

ブラウザで `http://localhost:8501` を開いてください。

## 各機能説明

| ページ | 機能 |
|---|---|
| ダッシュボード | 今月の稼働時間・面談件数・未完了 ToDo を KPI カードで表示。直近の日報・ToDo・面談を一覧表示。 |
| 案件管理 | 会社名・案件名・期間・メモの登録、キーワード検索、編集・削除。 |
| 面談管理 | 面談日・会社・案件・業務内容・勤怠・ステータス（通過/不通過/結果待ち）の登録・閲覧・削除。 |
| ToDo管理 | 案件紐づきタスク・期限・進捗（未着手/進行中/完了）の登録・更新・削除。 |
| 日報管理 | 日付・会社・案件・勤怠区分・勤務時間・業務内容の入力。登録と同時に CSV へ追記。 |
| レポート | 月報・週報の自動集計、勤務時間推移・勤怠区分分布のグラフ表示、CSV ダウンロード。 |
| 設定・データ管理 | CSV エクスポート（個別/ZIP）、CSV インポート（追記）、手動バックアップ、データリセット。 |

## データ仕様

すべてのデータは `data/` 以下の CSV に保存されます。DB 不要。  
起動時に `data/` が `backup/YYYYMMDD_HHMMSS/` へ自動コピーされます。
